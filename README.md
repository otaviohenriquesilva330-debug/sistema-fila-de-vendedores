# Sistema de Fila de Vendedores

Aplicação desenvolvida em Python com Tkinter para controle de fila rotativa de vendedores.

## Funcionalidades

- Cadastro de vendedores
- Fila rotativa (round-robin)
- Controle de atendimento
- Registro de conversão de vendas
- Remoção de vendedores

## Tecnologias

- Python
- Tkinter
- Git

## Objetivo

Projeto criado para prática de lógica, estruturas de dados e controle de estado.
